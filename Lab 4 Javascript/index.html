¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Semejanzas:

  • Utilizan sintaxis parecida estilo C
  • Son ampliamente usadas para aplicaciones web tanto del lado del cliente como del servidor
  • Cuentan con una librería Math

Diferencias:

  • JavaScript es interpretado mientras que Java es compilado
  • Java es de tipado riguroso, contrario a JavaScript
  • Java es un lenguaje de programación orientado a objetos mientras que JavaScript se basa en objetos
  • Java puede correr en cualquier máquina virtual, JavaScript puede correr en navegadores y ahora en servidores gracias a Node.js

¿Qué métodos tiene el objeto Date?

  • getDate(): regresa el día del mes
  • getDay(): regresa el día de la semana
  • getFullYear(): regresa las cuatro cifras del año actual
  • getHours(): regresa la hora actual
  • getMinutes(): regresa los minutos transcurridos de la hora actual
  • now(): regresa la fecha actual como el número de milisegundos transcurridos desde el 1 de enero de 1970

¿Qué métodos tienen los arreglos?

  • push(): inserta elemento al final del arreglo
  • pop(): elimina el elemento del final del arreglo
  • length(): regresa la longitud del arreglo
  • join(): regresa string de elementos del arreglo concatenados
  • indexof(): regresa el índice de un elemento dado

¿Cómo se declara una variable con alcance local dentro de una función?

Para declarar una variable con alcance local en una función se la debe instanciar por primera vez dentro de las llaves de la función sea con var o let. Let tiene alcance tan solo de bloque mientras que var tiene alcance de la función que la contiene.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Al utilizar una variable global se puede modificar su valor desde cualquier parte del código por lo que puede producir alteraciones inesperadas provenientes de otras partes del código
